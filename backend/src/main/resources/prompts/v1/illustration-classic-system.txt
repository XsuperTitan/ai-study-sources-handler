Generate a soft pastel information architecture infographic for a learning source package.
The image should feel like a clean, structured educational study chart, not an abstract memory poster.

Canvas and framing:
- The image service renders a square 2048 canvas. Compose a 16:9 horizontal infographic safely inside the square, with generous margins so it works as a homepage card cover.
- Use a white or very light paper background, subtle grid texture, thin rounded panel borders, and soft accent colors.

Layout:
- Place a central circular theme node in the middle with a simple hierarchy, flow, or concept icon.
- Surround it with 3-5 rounded information panels arranged left, right, top, and bottom.
- Use curved arrows to connect panels to the central node and make the reading path clear.

Text strategy:
- Short readable text is allowed and preferred when possible.
- Use only sparse, large, high-contrast text: one Chinese heading, optional English subtitle, 2-3 short labels per panel.
- Avoid long paragraphs, dense explanations, small captions, real code, and complex table text.
- The app still renders the reliable title and study summary outside the image, so image text is supportive rather than authoritative.

Visual language:
- Soft blue, mint green, lavender, and warm beige section accents.
- Clean hand-drawn marker icons, light watercolor header swatches, thin connector lines, balanced spacing, and consistent typography.
- Educational, gentle, readable, minimal, diagrammatic.

Avoid:
- Photorealism, 3D glass, dark sci-fi scenes, holograms, complex UI screenshots, code blocks, table stacks, crowded micro text, logos, watermarks, blurry text, garbled characters, and low-quality layout.
